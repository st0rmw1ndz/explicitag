An automatic MP4 rating tagger based on embedded lyrics.

## Usage

```
ExpliciTag 1.0.0
Copyright (C) 2024 frosty

  --help            Display this help screen.

  --version         Display version information.

  paths (pos. 0)    Required. List of file and/or directory paths.
```
